# Portfolio Website

A clean starter portfolio website using:
- HTML
- CSS
- JavaScript

## Project Structure

```
portfolio-website/
├── index.html
├── style.css
├── script.js
├── .gitignore
└── README.md
```

## Run Locally

Because projects are loaded from `projects.json`, run a local server:

1. Open terminal in `portfolio-website`.
2. Run:
   - `python -m http.server 5500`
3. Open:
   - `http://localhost:5500`
